# Installation

## Voraussetzungen
- Minecraft Server `1.21.11`
- Fabric Loader
- Fabric API

## Schritte
1. `mapswitch-<version>.jar` in den `mods/` Ordner legen.
2. Sicherstellen, dass `fabric-api` installiert ist.
3. Server starten.
4. Bei erstem Start wird Datapack-Bootstrap erstellt.
5. Server erneut starten, damit neue Dimensionen geladen werden.

## Wichtige Pfade
- Mod config: `config/mapswitch.json`
- Map root: `maps/`
