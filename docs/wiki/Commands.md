# Commands

## Verfuegbar
- `/mapswitch <mapname>`
- `/map switch <mapname>`

## Verhalten
- Der ausfuehrende Spieler wird in die Ziel-Map gewechselt.
- Spieler-Daten werden map-spezifisch geladen.
- Tab-Liste wird nach Map getrennt.
- Actionbar zeigt aktive Map.

## Beispiele
- `/mapswitch normal`
- `/map switch hardcore`
