# Configuration

Datei: `config/mapswitch.json`

## Beispiel
```json
{
  "default_map": "normal",
  "allowed_maps": ["normal", "hardcore", "farmwelt", "bauwelt"]
}
```

## Felder
- `default_map`: Fallback/Start-Map fuer neue Sessions.
- `allowed_maps`: Maps, fuer die Ordner/Bootstrap vorbereitet werden.
